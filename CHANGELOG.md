# Changelog

Detailed per-package history is generated by Changesets in `packages/ui/CHANGELOG.md`.

## 0.1.0

First public release of `@toris-dev/ui`.

- 16 components: Button, IconButton, Card, Input, Textarea, Select, Checkbox,
  Badge, Avatar, Spinner, Skeleton, Switch, Dialog, Toast, Tabs, Tooltip
  (+ Slot/Portal/VisuallyHidden primitives, useControllableState/useEscapeKey hooks)
- Design tokens (`--tori-*`), dark mode, ToriProvider with system theme tracking
- Shared field scaffolding (label/helper/error) across Input, Textarea, and Select
- Accessibility: WCAG-checked foreground tokens, keyboard-accessible interactive
  Card, stacked-dialog semantics, toast hover-pause, forced-colors focus,
  reduced-motion-safe spinner and skeleton
- Supply-chain-hardened release pipeline: SHA-pinned actions, `npm` environment
  gate, tag/version assertion, package-content verification
- 77 unit/interaction tests; monorepo with Vite playground and docs
